# Inventory

This project is a simple inventory management program developed in Python.  
The application allows users to register products by entering _the product name, quantity, and price_.

The system stores the registered products and allows users to view the list of added products. It also includes an option to exit the program.

## Features

| Option | Description |
|------|-------------|
| 1. Add product | Allows users to register new products |
| 2. List products | Displays the added products |
| 3. Exit | Closes the program |

> _The program calculates the **total cost of each product**, multiplying the quantity by the unit price._

## How to Run the Program

1. Make sure **Python** is installed on your computer.
2. Download or clone the project repository: git clone https://github.com/Lneiras/Inventario.git
3. Open the project folder.
4. Run the program with the following command: `python inventario.py`
